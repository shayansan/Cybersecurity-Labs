# Cybersecurity-Labs

A collection of hands-on cybersecurity labs, notes, and write-ups.

## Contents
- Networking fundamentals (TCP/IP, DNS, HTTP/HTTPS)
- Linux fundamentals (terminal, permissions, processes)
- Web security labs (OWASP Top 10, PortSwigger)
- Blue team basics (logs, monitoring, incident response)
- CTF write-ups and practice

## Goal
Build strong cybersecurity fundamentals and document practical learning progress.

## Tools used
- Wireshark
- Nmap
- Burp Suite
- Linux (Ubuntu/Kali)
- TryHackMe / PortSwigger Web Security Academy

## Disclaimer
All content is for educational and ethical purposes only.
